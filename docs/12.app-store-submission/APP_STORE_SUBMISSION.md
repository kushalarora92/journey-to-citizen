## üöÄ Deployment Steps

### 1. Deploy Web Version (for Privacy & Support URLs)

```bash
cd apps/frontend
npx expo export -p web
firebase deploy --only hosting
```

Verify URLs work:
- https://journey-to-citizen.web.app/privacy
- https://journey-to-citizen.web.app/support

### 2. Build iOS App

```bash
cd apps/frontend
eas build --platform ios --profile production
```

### 3. Build Android App

```bash
cd apps/frontend
eas build --platform android --profile production
```

### 4. Test Builds

- Download and install on test devices
- Verify all features work
- Check privacy and support links

### 5. Submit to App Store Connect

1. Login to App Store Connect
2. Create new app
3. Fill in all information from this guide
4. Upload screenshots
5. Upload build from EAS
6. Submit for review

### 6. Submit to Google Play Console

1. Login to Google Play Console
2. Create new app
3. Fill in all information from this guide
4. Upload screenshots and graphics
5. Upload build (AAB file)
6. Complete content rating questionnaire
7. Submit for review
